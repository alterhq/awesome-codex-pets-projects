# Awesome Codex Pets

A curated index of projects, tools, libraries, galleries, and experiments for Codex pets and compatible desktop companion workflows.

## Featured

- [OpenPets.sh](https://openpets.sh) - Largest Codex pet registry, with browsing, uploads, installs for Codex-compatible tools, and plugins for live desktop signals. Platform: web, macOS app. License: MIT.
- [Codex-Pets.net](https://codex-pets.net/) - Community hub powered by [codex-pet-share](https://github.com/portons/codex-pet-share), a self-hostable app for browsing, sharing, uploading, and installing animated pixel pet packages. Platform: web, TypeScript. License: MIT.

## Galleries And Registries

- [OpenPets.sh](https://openpets.sh) - Largest Codex pet registry, with browsing, uploads, installs for Codex-compatible tools, and plugins for live desktop signals. Platform: web, macOS app. License: MIT.
- [Petdex](https://petdex.crafter.run/) - Public gallery of animated companions for Codex with previews, collections, submissions, and install commands. Platform: web, desktop app. License: unknown.
- [Petdex Source](https://github.com/crafter-station/petdex) - Source repo for the Petdex public gallery, with pet browsing, animation previews, browser validation, submissions, and downloadable packs. Platform: web. License: MIT.
- [Codex Pets](https://codex-pet.org/codex-pets/) - Independent community directory for previewing, downloading, installing, and sharing Codex pet packages. Platform: web. License: unknown.
- [CodexPets.org](https://codexpets.org/) - Codex pet browser with package previews, hatch-pet brief builder, install guidance, and URL debugging tools. Platform: web. License: unknown.
- [codex-pet-share](https://github.com/portons/codex-pet-share) - Self-hostable web app behind Codex-Pets.net for sharing animated pixel pets, browsing a community gallery, multiplayer playground rooms, and one-line CLI installs. Platform: web, TypeScript, Cloudflare. License: MIT.
- [Codex PokéPets](https://github.com/dnnyngyen/codex-pokepets) - Large themed Codex pet gallery and installer covering 2D and 3D animated Pokémon sprites. Platform: shell, Codex custom pets. License: unknown.

## Desktop Companions And Players

- [OpenPets](https://github.com/alterhq/openpets) - Native macOS desktop pet app with MCP, CLI, assistant setup, release packaging, custom Codex pet support, and shared visible task state. Platform: macOS. License: MIT.
- [Clawd on Desk](https://github.com/rullerzhou-afk/clawd-on-desk) - Cross-platform desktop pet that watches Codex CLI, Claude Code, Cursor, opencode, and other AI coding agents. Platform: macOS, Windows, Linux. License: unknown.
- [Petty](https://github.com/LeslieLeung/petty) - Tauri desktop pet player for Codex-compatible pet assets, with built-in pet support and local pet-folder import. Platform: desktop, Tauri. License: unknown.
- [Petdex Dock](https://github.com/felipetodev/petdex-dock) - Electron desktop Codex pet dock that lives at the edge of the desktop. Platform: desktop, Electron. License: unknown.
- [Talking Pets](https://github.com/arata-ai-daisuki/talking-pets) - Local Codex pet add-on that monitors assistant speech from local conversation logs and reads it aloud through local TTS. Platform: macOS, Windows, Linux, Node.js, Swift. License: MIT.
- [Vibebud](https://github.com/Shellishack/vibebud) - Floating AI buddy for Codex, Claude Code, and other agents, with desktop, web, and Android surfaces for companion presence and notifications. Platform: web, macOS, Windows, Linux, Android. License: MIT.

## SDKs And Integrations

- [OpenPetsKit](https://github.com/alterhq/OpenPetsKit) - Embeddable Swift runtime for loading bundled pets, hosting desktop pet UI, and sending commands through the shared OpenPets local socket. Platform: macOS, Swift Package Manager. License: MIT.
- [agent-pet](https://github.com/gibbon/agent-pet) - Lightweight self-hosted animated companion widget for web apps, delivered as a vanilla DOM/widget library. Platform: web, TypeScript. License: Apache-2.0.
- [codex-pet-bridge](https://github.com/vcxzvfe/codex-pet-bridge) - Local-first bridge that turns AI agent status into pet-friendly events for desktop pets and XiaoZhi devices. Platform: JavaScript, MCP, webhooks, SSE. License: MIT.

## Creation And Validation Tools

- [codex-pet-generator-skill](https://github.com/BinaryFroggy/codex-pet-generator-skill) - Codex skill for generating compliant animated Codex pet spritesheets and `pet.json` packages from reference images. Platform: Codex skill, Python. License: unknown.
- [OpenAI hatch-pet skill](https://github.com/openai/skills/blob/main/skills/.curated/hatch-pet/SKILL.md) - Curated Codex skill workflow for generating, repairing, validating, previewing, and packaging Codex-compatible animated pets. Platform: Codex skill. License: MIT.

## Ambient Status And Experiments

- [OpenPets](https://github.com/alterhq/openpets) - Shared desktop pet status surface for agent progress, review prompts, completion states, failures, and lightweight animations. Platform: macOS. License: MIT.
- [Codex Watch Companion](https://github.com/b-nnett/codex-apple-watch) - Proof-of-concept Apple Watch companion and Mac bridge for keeping a Codex pet, task state, and voice-driven chat controls visible from the wrist. Platform: watchOS, macOS, Swift, JavaScript. License: unknown.
- [codex-pet-limit-rings](https://github.com/petergpt/codex-pet-limit-rings) - macOS companion overlay that follows the Codex pet and shows short-window and weekly usage-limit rings without patching Codex. Platform: macOS. License: MIT.
- [TokenPet](https://github.com/sugar301/TokenPet) - Windows desktop pet with a local HTTP proxy for token tracking and Codex/Petdex-format pet imports. Platform: Windows, .NET, WPF. License: MIT.
- [TamaCodex](https://github.com/Alichua/TamaCodex) - In-Codex pet experiment where a companion grows and evolves from local work signals, token usage, care events, and a macOS sidecar. Platform: macOS. License: MIT.
